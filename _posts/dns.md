## 记录的类型


| 记录类型                        | 意义                    |
| ------------------------------- | ----------------------- |
| A                               | WEB服务器的 `IPv4` 指向 |
| AAAA                            | 记录 IPV6解析记录       |
| CNAME (Canonical Name)          | 通常称别名解析          |
| URL (Uniform Resource Locator ) | 网址转发                |
| MX (Mail Exchange)              | 邮件交换记录            |
| NS（Name Server）               | 记录是域名服务器记录    |
